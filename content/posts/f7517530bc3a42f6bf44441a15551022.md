## 配置路径及说明

### 1. 最大亮度值定义

**文件位置**：\
`frameworks/base/core/res/res/values/config.xml`

**关键配置项**：

```xml
<integer name="config_screenBrightnessSettingMaximum">255</integer>
```

**说明**：

- 该值定义了系统**亮度调节范围的最大值**

- 通常范围为 0-255（Android 标准）

- OEM 厂商可能修改此值（如部分设备最大亮度为 4095）

### 2. 默认亮度值定义

**文件位置**：\
`frameworks/base/packages/SettingsProvider/res/values/defaults.xml`

**关键配置项**：

```xml
<integer name="def_screen_brightness">102</integer>
```

**说明**：

- 该值控制设备**首次启动时的默认亮度**

- 数值范围：0 \~ `config_screenBrightnessSettingMaximum`

- 102 约等于 40% 亮度（102/255≈0.4）

***

**示例计算**：

```text
102 / 255 × 400nit ≈ 160nit
```

***

## 修改步骤

### 1. 调整最大亮度范围（可选）

```xml
<!-- frameworks/base/core/res/res/values/config.xml -->
<integer name="config_screenBrightnessSettingMaximum">4095</integer>
```

### 2. 设置默认亮度值

```xml
<!-- frameworks/base/packages/SettingsProvider/res/values/defaults.xml -->
<!-- 推荐值：102(40%) ~ 153(60%) -->
<integer name="def_screen_brightness">128</integer> <!-- 50% 亮度 -->
```

### 3. OEM 厂商定制建议

在设备 overlay 中覆盖默认值：

```xml
<!-- device/<vendor>/<device>/overlay/frameworks/base/core/res/res/values/config.xml -->
<integer name="config_screenBrightnessSettingMaximum">4095</integer>

<!-- device/<vendor>/<device>/overlay/packages/SettingsProvider/res/values/config.xml -->
<integer name="def_screen_brightness">1024</integer> <!-- 4095的25% -->
```

***

## 验证方法

### 1. 查看当前亮度设置

```bash
adb shell settings get system screen_brightness
# 输出示例：128
```

### 2. 查看最大亮度值

```bash
adb shell settings get system screen_brightness_maximum
# 输出示例：255
```

### 3. 临时修改亮度（测试用）

```bash
# 设置亮度为 150（范围 0-255）
adb shell settings put system screen_brightness 150
```

***

## 注意事项

### 亮度范围建议

| 环境类型 | 推荐亮度值   | 对应百分比    |
| ---- | ------- | -------- |
| 夜间使用 | 51-102  | 20%-40%  |
| 室内常规 | 102-153 | 40%-60%  |
| 室外强光 | 204-255 | 80%-100% |

### 重要限制

1. **硬件限制**：\
   实际亮度不能超过显示面板的物理最大值

2. **节能要求**：\
   Android 兼容性定义文档(CDD)要求：

   ```text
   [7.2.4/H-1-1] 必须提供至少 100nit 的最低亮度
   [7.2.4/H-1-2] 必须提供至少 200nit 的室内典型亮度
   ```

3. **自动亮度依赖**：

   ```mermaid
   graph TD
       A[默认亮度] --> B[手动模式]
       C[自动亮度] --> D[亮度曲线配置文件]
   ```

   修改 `config_autoBrightnessLevels` 可调整自动亮度曲线

***

## 高级配置

### 亮度曲线调整

```xml
<!-- frameworks/base/core/res/res/values/config.xml -->
<integer-array name="config_autoBrightnessLevels">
    <item>10</item>  <!-- 光照阈值(lux) -->
    <item>50</item>
    <item>100</item>
    <item>1000</item>
</integer-array>

<integer-array name="config_autoBrightnessLcdBacklightValues">
    <item>20</item>   <!-- 对应亮度值 -->
    <item>60</item>
    <item>120</item>
    <item>220</item>
</integer-array>
```

### 恢复出厂设置

修改后需清除设置数据才能生效：

```bash
adb shell pm clear com.android.providers.settings
```

> **最佳实践**：\
> 对于终端用户，建议通过「设置 > 显示 > 亮度」手动调整，系统级修改适合 ROM 开发者或 OEM 厂商。
