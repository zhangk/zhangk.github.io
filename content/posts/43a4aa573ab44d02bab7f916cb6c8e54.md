## 🔍 核心命令

### 1. 搜索全部可用软件

```bash
sudo apt-cache search all
```

- 列出所有可用软件包（通常数千个）

- 按 `q` 键退出列表浏览模式

### 2. 关键词过滤搜索 (推荐)

```bash
sudo apt-cache search all | grep "关键词"
```

示例搜索 virtualbox：

```bash
sudo apt-cache search all | grep virtualbox
```

## 🌟 优化搜索技巧

### 使用正则表达式

```bash
# 搜索以"virtual"开头的软件
sudo apt-cache search all | grep "^virtual"

# 搜索包含"box"且以"vm"结尾的软件
sudo apt-cache search all | grep "box.*vm$"
```

### 显示完整描述

```bash
# 使用apt替代apt-cache（更友好格式）
apt search 关键词
```

示例：

```bash
apt search virtualbox
```

输出格式：

```text
pkg名/版本 架构 [仓库]
  描述摘要...
```

## 📂 高级搜索方法

### 按软件类型过滤

```bash
# 只搜索游戏类软件
apt search games | grep 关键词

# 查看所有分类
apt-cache dumpavail | grep Section:
```

### 查看软件详情

```bash
apt show 包名
```

示例：

```bash
apt show virtualbox-6.1
```

### 列出相关依赖

```bash
apt-cache depends 包名
```

## ⚠️ 注意事项

1. **权限要求**：

   - `apt-cache search` 无需sudo

   - `apt search` 无需sudo

   - `apt update` 需要sudo

2. **更新软件源**：

   ```bash
   sudo apt update  # 先更新源再搜索
   ```

3. **搜索限制**：

   ```bash
   # 限制返回数量
   apt-cache search --names-only 关键词 | head -n 20
   ```

## 📊 实用命令对照表

| 功能   | 传统命令                 | 新版命令           |
| ---- | -------------------- | -------------- |
| 搜索软件 | `apt-cache search`   | `apt search`   |
| 查看详情 | `apt-cache show`     | `apt show`     |
| 列出依赖 | `apt-cache depends`  | `apt depends`  |
| 反向依赖 | `apt-cache rdepends` | `apt rdepends` |

> 💡 **技巧**：使用 `Tab` 键自动补全包名，如 `apt install virtualbox<Tab>` 会显示所有匹配项
