## 1. 获取订阅文件

- **官网地址**\
  <https://dog1.ssrdog7.com/#/signin>
- **操作步骤**：
  1. 登录后进入「儀表盤」
  2. 打开「訂閱中心」
  3. 点击「操作」→「複製備用訂閱地址」

## 2. 安装客户端

- **下载地址**\
  [](https://github.com/LibNyanpasu/clash-nyanpasu/releases/download/v1.6.1/clash-nyanpasu_1.6.1_amd64.AppImage)在github(<https://github.com/clash-verge-rev/clash-verge-rev)上下载最新版本的.deb，如：> [https://github.com/clash-verge-rev/clash-verge-rev/releases/tag/v2.4.0/Clash.Verge\_2.4.0\_amd64.deb](https://github.com/clash-verge-rev/clash-verge-rev/releases/tag/v2.4.0)

- **安装命令**：

  ```bash
  sudo dpkg -i Clash.Verge_2.4.0_amd64.deb
  ```

## 3. 配置客户端

1. 左侧点击「订阅」

2. 把第一步复制的訂閱地址粘贴到上方的输入框中，点击"导入"按钮

## 4. 启用代理

1. 左侧点击「设置」

2. 虚拟网卡模式中点击"安装服务"，安装后开启。
