## 下载安装文件

从GitHub下载RustDesk服务器端程序：

```bash
wget https://github.com/rustdesk/rustdesk-server/releases/download/1.1.14/rustdesk-server-hbbr_1.1.14_amd64.deb
wget https://github.com/rustdesk/rustdesk-server/releases/download/1.1.14/rustdesk-server-hbbs_1.1.14_amd64.deb
```

## 安装服务

使用dpkg安装下载的deb包：

sudo dpkg -i rustdesk-server-hbbr\_1.1.14\_amd64.deb rustdesk-server-hbbs\_1.1.14\_amd64.deb

## 获取公钥

查看并复制公钥，用于客户端配置：

cat /var/lib/rustdesk-server/id\_ed25519.pub

## 客户端配置

在RustDesk客户端中设置：

- **Key**: 粘贴上面获取的公钥

- **ID服务器**: 填写阿里云服务器的公网IP地址

- **中继服务器**: 填写阿里云服务器的公网IP地址

## 防火墙配置

在阿里云安全组中放行以下端口：

**TCP协议端口：**

- 21114

- 21115

- 21116

- 21118

- 21119

**UDP协议端口：**

- 21116 (需要同时开启TCP和UDP)

## 服务管理

安装完成后，系统会自动启动以下服务：

- rustdesk-hbbs (RustDesk ID服务器)

- rustdesk-hbbr (RustDesk中继服务器)

可以使用systemctl管理服务状态：

sudo systemctl status rustdesk-hbbs\
sudo systemctl status rustdesk-hbbr

## 注意事项

1. 确保服务器有足够的带宽支持远程连接

2. 建议定期更新RustDesk服务器版本

3. 重要数据请做好备份
