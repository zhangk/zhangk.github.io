## 📦 操作流程

### 1. 查看已安装软件列表

```bash
dpkg --list
```

- 按首字母排序显示所有已安装软件

- 滚动查找目标软件名（如 `polipo`）

### 2. 执行卸载命令

```bash
# 删除软件及其配置文件（彻底清除）
sudo apt-get --purge remove 包名

# 仅删除软件（保留配置）
sudo apt-get remove 包名
```

示例（卸载 polipo）：

```bash
sudo apt-get --purge remove polipo
```

### 3. 确认操作

```bash
您希望继续执行吗？ [Y/n]
```

- 输入 `y` 后按回车确认

- 输入密码时不会显示字符（盲输）

### 4. 自动清理残留

```bash
sudo apt-get autoremove  # 删除不再需要的依赖包
sudo apt-get clean       # 清理软件包缓存
```

## ⚠️ 注意事项

1. 需要管理员权限（使用 `sudo`）

2. 精确匹配软件包名（区分大小写）

3. `--purge` 会彻底删除配置数据（谨慎使用）

4. 关键系统组件勿随意卸载

## 🔍 辅助命令

```bash
# 搜索软件包
apt search 关键词

# 查看软件详情
apt show 包名

# 修复依赖问题
sudo apt-get install -f
```

> 💡 提示：卸载浏览器等复杂软件时，建议先关闭相关进程

```bash
# 强制结束相关进程
sudo pkill -f 进程名
```
