## 命令格式

```bash
grep --exclude-dir='目录名' -rn "搜索内容" /搜索路径/
```

## 示例

搜索 `/path/to/search` 中所有文件，查找包含 `pattern` 的内容，但跳过 `exclude_dir` 目录：

bash

grep --exclude-dir='exclude\_dir' -rn "pattern" /path/to/search

## 参数说明

- `--exclude-dir='目录名'`：忽略指定的目录（支持通配符）

- `-r` 或 `-R`：递归搜索子目录

- `-n`：显示匹配行的行号

- `"pattern"`：要搜索的模式或字符串

## 其他排除选项

```bash
# 排除多个目录  
grep --exclude-dir={'dir1','dir2'} -rn "pattern" /path/
```

```bash
# 排除特定扩展名的文件  
grep --exclude='\*.log' -rn "pattern" /path/
```

```bash
# 从文件中读取排除模式  
grep --exclude-from=exclude_list.txt -rn "pattern" /path/
```

## 注意事项

1. `--exclude-dir` 参数在较新版本的 grep 中可用（GNU grep 2.5.1+）

2. 目录名区分大小写

3. 可以使用相对路径或绝对路径

## 常见用例

- 搜索代码时跳过 `.git` 目录

- 搜索时跳过日志目录

- 忽略编译生成的文件目录
