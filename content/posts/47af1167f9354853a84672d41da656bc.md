## 操作说明

此流程用于解锁MTK（联发科）芯片平台Android 15设备的Bootloader，以便进行刷机、Root等高级操作。**注意：解锁会清除设备内所有数据，请提前备份。**

## 解锁步骤

1. **开启OEM解锁**
   在设置->开发者选项中把OEM解锁选项开启。
2. **重启到Bootloader模式**\
   在已开启USB调试的设备上执行

```bash
adb reboot bootloader
```

**执行解锁命令**\
设备进入fastboot模式后，执行解锁指令：

```bash
fastboot flashing unlock
```

- *部分设备可能需要使用 `fastboot oem unlock`，请根据屏幕提示操作*

- **重启设备**\
  解锁完成后重启设备：

```bash
fastboot reboot
```

## ⚠️ 重要注意事项

- **数据丢失**：解锁过程会自动恢复出厂设置，请务必备份重要资料

- **保修影响**：部分厂商设备解锁后可能影响保修服务

- **风险提示**：错误操作可能导致设备变砖，请确认设备型号和平台支持

- **驱动准备**：操作前请确保已安装正确的ADB和Fastboot驱动
