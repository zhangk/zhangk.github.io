## 🧪 测试验证方法

```bash

adb logcat -s BatteryPercentage

# 模拟电池变化
adb shell dumpsys battery set level 50
```

## ⚙️ 编译部署步骤

1. 重新编译SystemUI模块:

   ```bash
   make SystemUI
   ```

2. 推送更新文件:

   ```bash
   adb remount
   adb push out/target/product/<device>/system/priv-app/SystemUI/SystemUI.apk /system/priv-app/SystemUI/
   ```

3. 重启SystemUI:

   ```bash
   adb shell pkill com.android.systemui
   ```

## 📝 注意事项

1. **尺寸资源定义**：确保在`res/values/dimens.xml`中添加：

   ```xml
   <dimen name="battery_level_text_size">12sp</dimen>
   <dimen name="battery_level_padding_end">4dp</dimen>
   ```

2. **多语言支持**：百分比格式会自动适配系统语言

3. **布局优化**：如果位置不理想，可调整：

   ```xml
   android:layout_marginStart="4dp"
   android:layout_marginEnd="2dp"
   ```

4. **充电状态处理**：如需充电时显示不同样式，可添加：

   ```java
   if (pluggedIn) {
       mBatteryLevel.setTextColor(Color.GREEN);
   } else {
       mBatteryLevel.setTextColor(Color.WHITE);
   }
   ```

> 💡 **提示**：此修改适用于MTK平台的Android 5.1系统，其他平台可能需要调整布局参数
