## 安装步骤

### 1. 安装输入法组件

打开终端执行：

```bash
sudo apt update && sudo apt install fcitx5 fcitx5-chinese-addons fcitx5-frontend-qt5
```

### 2. 重启系统

```bash
sudo reboot
```

### 3. 配置输入法

系统重启后：

1. 打开终端运行配置工具：

```bash
fcitx5-configtool
```

2. 点击界面左下角 **+** 按钮

3. 取消勾选 *Only Show Current Language*

4. 搜索并添加：

   - WubiPinyin (五笔拼音)

   - Pinyin (拼音备用)

### 6. 设置开机自启

在KDE设置中：

1. 进入 *系统设置 > 开机和关机 > 自动启动*

2. 添加程序：`fcitx5`

## 使用说明

- 切换输入法：`Ctrl` + `Space`

- 切换中英文：`Shift`

- 候选词翻页：`,` 和 `.`

## 故障排查

1. 输入法不生效：

```bash
fcitx5 -d --replace
```

2. 检查安装状态：

```bash
fcitx5 -v
```

> **提示**：安装后首次使用可能需要重新登录才能生效\
> **推荐**：在fcitx配置中启用云拼音和联想功能提升输入体验

## 功能特点

- 五笔拼音混合输入
- 动态词频调整
- 支持简繁体切换
- 可自定义快捷键
- 低资源占用

> 此配置同时适用于其他基于Debian的发行版（Ubuntu/Kubuntu等），KDE Plasma 5环境下测试通过
