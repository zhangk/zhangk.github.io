## 🔧 操作步骤

### 1. 启动TCP调试模式

```bash
adb tcpip 5555
```

**注意**：

- 需先用数据线连接设备与电脑

- 成功执行后会显示 `restarting in TCP mode port: 5555`

### 2. 连接设备

```bash
adb connect <设备IP>
```

**示例**：\
`adb connect 192.168.1.100`

## ⚠️ 常见错误处理

### 报错信息：

`missing port in specification: tcp:`

### ✅ 解决方案：

在IP地址后显式添加端口号：

```bash
adb connect <设备IP>:5555
```

## 💡 连接成功验证

执行命令后出现 `connected to <IP>:5555` 表示连接成功，此时可断开数据线进行无线调试。

## 注意事项：

- 确保设备与电脑在同一局域网
- 首次使用需开启设备的USB调试模式
- 端口号5555是通用默认端口，如被占用可更换其他端口
- 重新连接设备无需重复步骤1，直接执行adb connect即可
