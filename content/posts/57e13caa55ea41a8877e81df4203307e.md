## 修改目的

将 MTP 模式连接电脑时显示的设备型号名称修改为自定义名称 "Smartab"

## 修改位置

```diff
# 文件路径
frameworks/av/media/mtp/MtpServer.cpp

# 修改内容
@@ -494,7 +494,7 @@ MtpResponseCode MtpServer::doGetDeviceInfo() {
     mData.putAUInt16(playbackFormats);  // Playback Formats
 
     mData.putString(mDeviceInfoManufacturer); // Manufacturer
-    mData.putString(mDeviceInfoModel); // Model
+    mData.putString("Smartab"); // Model
     mData.putString(mDeviceInfoDeviceVersion); // Device Version
     mData.putString(mDeviceInfoSerialNumber); // Serial Number
```

## 效果说明

| 连接模式   | 原始显示   | 修改后显示   |
| ------ | ------ | ------- |
| MTP    | 设备实际型号 | Smartab |
| PTP    | 设备实际型号 | 不变      |
| USB大容量 | 设备实际型号 | 不变      |

## 影响范围

1. **Windows 资源管理器**：

   - 设备名称将显示为 "Smartab"

   - 盘符标签保持原样
2. **Android 文件传输协议**：

   ```mermaid
   sequenceDiagram
       PC->>Device: 请求设备信息 (GetDeviceInfo)
       Device->>PC: 返回制造商(Manufacturer)
       Device->>PC: 返回型号(Model) = "Smartab"
       Device->>PC: 返回版本(Device Version)

   ```

## 编译部署步骤

1. 修改 MtpServer.cpp 文件

2. 重新编译 MTP 模块：

   ```bash
   mmm frameworks/av/media/mtp/
   ```

3. 替换库文件：

   ```bash
   adb push out/target/product/<device>/system/lib/libmtp.so /system/lib/
   adb reboot
   ```

## 验证方法

1. 连接设备到电脑（选择 MTP 模式）

2. 在 Windows 中查看设备名称：

   - 打开设备管理器 → 便携设备

   - 应显示 "Smartab"

3. 通过代码验证：

   ```bash
   adb shell dumpsys media.mtp | grep "Device Info"
   ```

## 注意事项

1. **多位置定义**：\
   设备信息可能来自多个位置：

   ```c
   // 原始值来源
   mDeviceInfoModel = getenv("PRODUCT_MODEL"); 
   // 或来自 build.prop
   ```

2. **完整设备信息**：\
   建议同时修改以下属性确保一致性：

   ```cpp
   mData.putString("Smartab Inc."); // 制造商修改
   mData.putString("V2.0");         // 设备版本修改
   ```

3. **恢复默认**：\
   如需恢复原始行为：

   ```cpp
   mData.putString(mDeviceInfoModel); // 恢复原始型号
   ```

4. **品牌合规性**：\
   修改设备型号名称需符合当地法规要求：

   - FCC 认证（美国）

   - CE 认证（欧洲）

   - SRRC 认证（中国）
